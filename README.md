# wedding-bootstrap

## Description

A single-page wedding website built for Kayleigh and Justin's wedding, providing comprehensive event information, registry links, and an integrated photo gallery.

## Key Features
- **Responsive Design** - Mobile-first approach using Bootstrap framework for optimal viewing across all devices
- **Single Page Application** - Smooth scrolling navigation with dynamic navbar shrinking on scroll
- **Custom Branding** - Personalized logo and color scheme with custom CSS styling
- **Interactive Navigation** - Collapsible mobile menu with smooth scroll-spy functionality
- **External Integrations** - Connected to Google Forms for song requests and external registry links
- **Static Deployment** - Lightweight, fast-loading site optimized for GitHub Pages hosting

Languages used:
- HTML
- CSS
- JavaScript

This stack was specifically selected for simplicity and ease of deployment as a single-page static website.

## Installation
To run locally:
In the desired directory, enter **git clone https://github.com/kkowatsch/wedding-bootstrap.git** in the terminal and hit the **return** key on the keyboard. 

## Usage
To run locally, simply open `index.html` in your web browser, or serve it with a local web server:
```bash
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000` in your browser.

## Deployment
The website can be deployed to GitHub Pages or any static hosting service. The CNAME file is configured for custom domain hosting.
